# About the Author

**`shared-features` is built and maintained by [Ahsan Mahmood](https://aoneahsan.com)** — a full-stack and mobile engineer who builds React, Capacitor, and developer-tooling products, and publishes open-source packages under the npm handle [`aoneahsan`](https://www.npmjs.com/~aoneahsan).

`shared-features` came out of running a whole portfolio of apps and getting tired of copy-pasting the same footer, the same "try my other app" promo, and the same maintenance banner into every project. One shared package, one admin panel, one source of truth.
